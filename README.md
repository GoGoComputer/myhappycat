
## 🎮 시작하기

### 첫 번째 단계

1. **촌장님 만나기** - 마을에서 촌장을 찾아 인사하세요
2. **슬라임 퇴치** - 첫 전투로 슬라임 5마리를 처치하세요
3. **장비 받기** - 대장장이에게서 기본 장비를 받으세요
4. **직업 선택** - 레벨 10이 되면 첫 전직을 할 수 있습니다!

### 초보자 튜토리얼 라인

- 사냥 → 장비 → 강화 → 던전 순서로 진행됩니다.
- 진행 확인: `.튜토리얼`
- 다음 목표 추천: `.추천루트`
- 짧고 보상이 체감되는 던전: `.초보던전`

### 성장 루트 (스토리 연계)

에테르니아의 균열이 열리기 전, 모험가 조합은 **세 단계의 시험**을 마련했습니다.

1) **숲의 첫 시험** (Lv.1~15)
  - 내적 성장: 튜토리얼 퀘스트 라인
  - 콘텐츠: `.초보던전`

2) **정예의 증명** (Lv.16~35)
  - 현자들의 요청으로 정예 훈련장에 입장
  - 콘텐츠: `.중급던전`, `.정예사냥`

3) **심연의 대비** (Lv.36~)
  - 균열의 그림자에 맞서는 준비
  - 콘텐츠: `.변이던전`, `.시즌던전`, `.도전모드`

### 초보자 스탯 가이드

| 스탯 | 설명 | 중요 직업 |
|-----|------|----------|
| HP | 생명력 - 0이 되면 사망 | 전사, 기사 |
| MP | 마나 - 스킬 사용에 필요 | 마법사, 힐러 |
| ATT | 공격력 - 물리 데미지 결정 | 전사, 도적, 궁수 |
| DEF | 방어력 - 받는 데미지 감소 | 기사, 팔라딘 |
| MAGIC | 마법력 - 마법 데미지 결정 | 마법사 |
| SPEED | 속도 - 행동 순서, 회피에 영향 | 도적, 궁수 |

### 타로 집중 시스템

`.타로집중` 명령어로 하루에 한 번 특정 속성에 보너스를 집중시킬 수 있습니다.

| 타입 | 명령어 예시 | 효과 |
|------|------------|------|
| 공격 | `.타로집중 공격` | 공격력 x1.04 |
| 방어 | `.타로집중 방어` | 방어력 x1.04 |
| 체력 | `.타로집중 체력` | HP +70 |
| 드랍 | `.타로집중 드랍` | 드랍률 +3% |
| 경험 | `.타로집중 경험` | 경험치 +3% |
| 골드 | `.타로집중 골드` | 골드 +3% |
| 치유 | `.타로집중 치유` | 치유량 +3% |

```
💡 타로 집중 전략
• 레벨업 중: .타로집중 경험
• 골드 파밍: .타로집중 골드
• 희귀템 노리기: .타로집중 드랍
• 보스전 준비: .타로집중 공격 또는 .타로집중 방어
• 파티 힐러: .타로집중 치유
```

### 타로 보너스 적용 범위

타로 카드로 얻은 보너스는 다음 항목에 자동 적용됩니다:

- **전투**: 공격력/방어력/HP 보너스
- **사냥 보상**: 경험치/골드/드랍률 보너스
- **회복**: 치유량 보너스 (`.휴식`, 힐 스킬 등)

```
📊 보너스 계산 예시
기본 공격력 100, 마법사(x1.08) + 힘(x1.06) + 공격 집중(x1.04)
→ 100 × 1.08 × 1.06 × 1.04 = 119 (약 19% 증가)
```

### 타로 플레이 팁

1. **매일 뽑기 필수** - 무료 버프이니 꼭 챙기세요
2. **집중 선택은 신중히** - 하루에 한 번만 가능
3. **플레이 스타일에 맞게** - 사냥 위주면 경험/드랍, 보스전이면 공격/방어
4. **메이저 카드 노리기** - 마이너보다 효과가 강력합니다
5. **파티 역할 고려** - 힐러면 치유 집중, 탱커면 방어/체력 집중

## 📲 카카오톡 메신저봇 R 초기 세팅

아래는 **카카오톡 메신저봇 R**에서 이 봇을 처음 구동할 때 필요한 기본 흐름입니다. 환경마다 메뉴 위치가 조금 다를 수 있으니, 핵심 흐름만 따라가면 됩니다.

### 1) 앱 설치 및 기본 권한
1. **카카오톡 메신저봇 R** 설치
2. 앱 실행 후 **필수 권한 허용**
  - 알림 접근, 접근성(또는 알림 읽기/응답) 권한

### 2) 봇 스크립트 연결
1. 앱에서 **새 봇** 생성
2. 스크립트 편집기에 **dotForestall.js** 전체를 붙여넣기
3. **저장** 후 **빌드/적용** 버튼으로 반영

### 3) 응답 방식 확인
- 메신저봇 R은 기본적으로 `response(room, msg, sender, isGroupChat, replier, imageDB, packageName)` 함수를 호출합니다.
- 메시지는 점(`.`)으로 시작해야 명령어로 인식됩니다.

### 4) 테스트 체크리스트
1. 개인/단체 채팅에서 `.도움말` 입력
2. `.시작` 또는 `.캐릭생성`으로 캐릭터 생성 확인
3. `.정보`로 상태 표시 정상 여부 확인
4. `.사냥` → `.공격`으로 전투 정상 확인

### 5) 저장 경로/데이터
- 기본 저장은 앱이 제공하는 **전용 데이터 폴더**를 사용합니다.
- 저장/로드에 문제가 있으면 **앱 권한(저장소)**를 확인하세요.

### 6) 운영 팁
- 스크립트 수정 후에는 **저장 → 빌드/적용**을 반드시 수행하세요.
- 런타임 오류가 날 경우 **오류 메시지 + 줄번호**를 확인하면 빠르게 해결할 수 있습니다.

### 7) 최소 구동 명령어
- `.도움말`
- `.시작`
- `.정보`
- `.사냥`

이제 바로 모험을 시작할 수 있습니다.
