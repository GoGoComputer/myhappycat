## 🎮 시작하기

### 첫 번째 단계

1. **촌장님 만나기** - 마을에서 촌장을 찾아 인사하세요
2. **슬라임 퇴치** - 첫 전투로 슬라임 5마리를 처치하세요
3. **장비 받기** - 대장장이에게서 기본 장비를 받으세요
4. **직업 선택** - 레벨 10이 되면 첫 전직을 할 수 있습니다!

### 초보자 스탯 가이드

| 스탯 | 설명 | 중요 직업 |
|-----|------|----------|
| HP | 생명력 - 0이 되면 사망 | 전사, 기사 |
| MP | 마나 - 스킬 사용에 필요 | 마법사, 힐러 |
| ATT | 공격력 - 물리 데미지 결정 | 전사, 도적, 궁수 |
| DEF | 방어력 - 받는 데미지 감소 | 기사, 팔라딘 |
| MAGIC | 마법력 - 마법 데미지 결정 | 마법사 |
| SPEED | 속도 - 행동 순서, 회피에 영향 | 도적, 궁수 |

## 🧭 명령어 가이드

**모든 명령어는 점(`.`)으로 시작합니다.**

### 1) 기본/정보
- `.시작 [직업]` : 캐릭터 생성 (직업 미입력 시 초보자)
- `.캐릭생성 <슬롯> [직업]` : 지정 슬롯에 캐릭터 생성
- `.캐릭선택 <슬롯>` : 슬롯 캐릭터 선택
- `.캐릭목록` : 슬롯 목록 확인
- `.캐릭삭제 <슬롯> 확인` : 슬롯 캐릭터 삭제(보호)
- `.정보` : 캐릭터 정보 확인
- `.도움말` 또는 `.명령어` : 전체 명령어 목록
- `.랭킹` : 랭킹 확인

### 2) 전투
- `.사냥` : 몬스터 사냥 시작
- `.공격` / `.공경` : 전투 진행
- `.도망` : 전투 이탈
- `.스킬목록` : 사용 가능한 스킬 목록
- `.스킬 <이름>` : 스킬 사용

### 3) 회복/보상
- `.휴식` : 휴식 (쿨타임 있음)
- `.일일보상` : 일일 보상 수령

### 4) 파티/레이드
- `.파티생성` / `.레이드생성` : 파티/레이드 생성
- `.파티초대 <유저>` / `.파티수락 <유저>` : 초대/수락
- `.파티나가기` : 파티 탈퇴
- `.파티역할 <탱/힐/딜>` : 역할 지정
- `.파티정보` : 파티 구성 확인

### 5) 던전
- `.던전목록` : 던전 목록
- `.던전 <이름> [노말/영웅/신화]` : 던전 입장
- `.인던 <이름> [난이도]` : 인스턴스 던전 입장
- `.인던로그` / `.레이드로그` : 로그 보기
- `.로그탭 인던|레이드 보스|난이도 [페이지]` : 로그 페이지
- `.로그상세 인던|레이드 [난이도=] [보스=]` : 로그 상세

### 6) PVP/전장
- `.결투 <유저>` / `.결투수락 <유저>` : 결투 신청/수락
- `.전장참가` : 전장 매칭
- `.전장맵` : 전장 맵 확인

- `.시즌랭킹` / `.시즌보상` : 시즌 랭킹/보상

### 7) 채집/전문기술/제작

### 🌲 묘냥의 숲 (Lv.1-15) - 시작 지역

> *"평화로운 숲 지역. 모험가들이 처음 여정을 시작하는 곳이다."*

| 세부 지역 | 레벨 | 출현 몬스터 |
|----------|------|------------|
| 평화의 초원 | 1-5 | 슬라임, 쥐, 토끼 |
| 어둠의 숲길 | 5-10 | 고블린, 늑대, 박쥐 |
| 안개 늪지 | 8-12 | 거미, 뱀, 습지괴물 |
| 고대 유적터 | 10-15 | 스켈레톤, 고스트, 골렘조각 |

**🏠 주요 NPC**
- 👴 **현자 알드윈** - 퀘스트, 조언
- 🔨 **대장장이 그론드** - 장비 제작/강화
- 🛒 **상인 마르코** - 물품 구매/판매
- 🏹 **사냥꾼 라이카** - 펫 관련 퀘스트

**⚔️ 던전**
- `고블린 동굴` - 고블린두목 처치
- `거미 둥지` - 초보자용 던전
- `잊혀진 지하실` - 언데드 출현

**💎 자원**: 약초, 나무, 철광석, 가죽

## 📲 카카오톡 메신저봇 R 초기 세팅

아래는 **카카오톡 메신저봇 R**에서 이 봇을 처음 구동할 때 필요한 기본 흐름입니다. 환경마다 메뉴 위치가 조금 다를 수 있으니, 핵심 흐름만 따라가면 됩니다.

### 1) 앱 설치 및 기본 권한
1. **카카오톡 메신저봇 R** 설치
2. 앱 실행 후 **필수 권한 허용**
  - 알림 접근, 접근성(또는 알림 읽기/응답) 권한

### 2) 봇 스크립트 연결
1. 앱에서 **새 봇** 생성
2. 스크립트 편집기에 **dotForestall.js** 전체를 붙여넣기
3. **저장** 후 **빌드/적용** 버튼으로 반영

### 3) 응답 방식 확인
- 메신저봇 R은 기본적으로 `response(room, msg, sender, isGroupChat, replier, imageDB, packageName)` 함수를 호출합니다.
- 메시지는 점(`.`)으로 시작해야 명령어로 인식됩니다.

### 4) 테스트 체크리스트
1. 개인/단체 채팅에서 `.도움말` 입력
2. `.시작` 또는 `.캐릭생성`으로 캐릭터 생성 확인
3. `.정보`로 상태 표시 정상 여부 확인
4. `.사냥` → `.공격`으로 전투 정상 확인

### 5) 저장 경로/데이터
- 기본 저장은 앱이 제공하는 **전용 데이터 폴더**를 사용합니다.
- 저장/로드에 문제가 있으면 **앱 권한(저장소)**를 확인하세요.

### 6) 운영 팁
- 스크립트 수정 후에는 **저장 → 빌드/적용**을 반드시 수행하세요.
- 런타임 오류가 날 경우 **오류 메시지 + 줄번호**를 확인하면 빠르게 해결할 수 있습니다.

### 7) 최소 구동 명령어
- `.도움말`
- `.시작`
- `.정보`
- `.사냥`

이제 바로 모험을 시작할 수 있습니다.
